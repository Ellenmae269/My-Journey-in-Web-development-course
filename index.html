Web Systems and Technology

IT326

This course covers the fundamentals of web development, including HTML, CSS, JavaScript, and modern web design principles. Students will learn to build responsive, interactive websites.

Semester: 2nd Semester Student: Ellen Mae J. Borja Activities: 5
Basic HTML Structure Text Color & Font Styling Headings & Text Formatting Superscript & Subscript Ordered & Unordered Lists Quotes & Preformatted Text Definition Lists Abbreviations & Acronyms Hyperlinks & Navigation Anchor Links Images & Media Address Formatting
  • Build a basic webpage from scratch using proper HTML structure including head and body sections
  • Apply text colors, font families, and different heading sizes to style content on a webpage
  • Use text formatting tags such as bold, italic, underline, superscript, and subscript correctly
  • Create both ordered and unordered lists to organize and present information clearly
  • Use semantic HTML elements like blockquote, pre, del, ins, address, and definition lists properly
  • Display abbreviations and acronyms with their full meaning using the abbr tag
  • Create hyperlinks to external websites and open them in a new browser window
  • Build in-page anchor navigation that jumps between the top and bottom of a page
  • Display images on a webpage with proper titles, dimensions, and borders
  • Link images to external websites and to themselves for full browser display
